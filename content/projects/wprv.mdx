# Praise Radio Vision Networks

This repository contains the source code for the WPRV Android app.

If you have any issues or requests, please see [issues](https://github.com/PatinoBrian/PraiseRadioVisionNetworks/issues) to report any bugs or feature requests.

--------------------------
## Build
The build requires [Android Studio](https://developer.android.com/sdk/installing/studio.html) and the [Android SDK](http://developer.android.com/sdk/index.html) installed in your developmental environment. You must also set `ANDROID_HOME` environmental variable to the location of your Android SDK.

Once complete:
* Open this project with Android Studio
* Go to Run>Run'app'
* Either connect an Android device to your computer or start a virtual device

You have installed this app from the Google Play store, please uninstall the app and then install this build. This is an Android security feature that blocks overwriting apps with different signed keys.
